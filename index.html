Public link

Public links always link to the latest version of a file.

Public links look like: https://${username}.files.fission.name/p/${path_to_file}

Archival Link

Archival links permanently link to one version of the file.

When you update a file, the content identifier (root CID) for the user's filesystem gets updated. You can use the root CID and the file path to get a permanent link to the content.

These links look like https://ipfs.runfission.com/ipfs/${rootCid}/p/${path_to_file}

Older Versions

Webnative keeps Versions of each file. You can link to older versions of the file by version CID.

These links look like https://ipfs.runfission.com/ipfs/${version.cid}/userland
